# The Pawn

Static browser presentation of locally extracted original text with updated scene graphics.

Open `original/index.html`, or use the GitHub Pages root URL after deployment.

The deployed site includes the browser app and web-sized updated scene images only. Raw disk images, intermediate MAG/GFX files, PRG extracts, local extraction scripts, and the old clean-room remake are intentionally not part of the published site.

Rights note: the original game text and images may be copyrighted. Publish or share this repository only where you have the rights or permission to distribute those assets.
