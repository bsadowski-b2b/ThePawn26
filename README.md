# The Pawn Original Asset Edition

Static browser presentation of locally extracted Commodore 64 image and text assets.

Open `original/index.html`, or use the GitHub Pages root URL after deployment.

The deployed site includes the browser app and rendered PNG images only. Raw disk images, intermediate MAG/GFX files, PRG extracts, and local extraction scripts are intentionally not part of the published site.

Rights note: the original game text and images may be copyrighted. Publish or share this repository only where you have the rights or permission to distribute those assets.
