# The Pawn - Original Asset Edition

This static browser edition uses locally extracted C64 assets:

- `assets.js` is a prebuilt browser bundle generated from the local text extraction.
- The 30 illustrations are loaded from `../images/png/`.
- The default view shows the original illustration, transcript, and command prompt.
- The command layer maps a small early-world route onto decoded original string indices; helper controls and search live in the folded Tools drawer.

It is not a recovered C64 VM or a full recreation of the original game logic. It is a browsable/playable presentation of the original extracted image set and decoded text table.
