# The Pawn

This static browser edition is a manual recreation workspace using locally extracted original text with updated scene graphics:

- `assets.js` is a prebuilt browser bundle generated from the local text extraction.
- Updated scene illustrations are loaded from `../images/updated/`.
- The default view shows updated scene artwork, transcript, and command prompt.
- The command layer maps the confirmed early-world route onto decoded original string indices.
- Additional room-like original strings are seeded as draft scenes so the map can be filled in manually.
- The Tools drawer includes a debug menu for jumping to mapped and draft scenes, testing scene-specific interactions, and reviewing text responses linked to the current scene.

It is not a recovered C64 VM. The full manual recreation will require filling in exits, object state, inventory rules, score rules, and conditional puzzle responses.
