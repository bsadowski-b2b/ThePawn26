# The Pawn

This static browser edition uses locally extracted original text with updated scene graphics:

- `assets.js` is a prebuilt browser bundle generated from the local text extraction.
- Updated scene illustrations are loaded from `../images/updated/`.
- The default view shows updated scene artwork, transcript, and command prompt.
- The command layer maps a small early-world route onto decoded original string indices; helper controls and search live in the folded Tools drawer.
- The Tools drawer includes a debug menu for jumping to scenes and testing scene-specific messages and interactions.

It is not a recovered C64 VM or a full recreation of the original game logic. It is a browsable/playable presentation of the decoded text table with updated artwork.
